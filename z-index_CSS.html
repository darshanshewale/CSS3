CSS z-Index property


    The Z-indes property is used to displace element on the z-axis i.e in or outof the screen it is used to define the order of elements if they overlap on each other
    if they overlap on each other.
    
    Property values:
    @Auto: the stack order is equal to that the parent(default)

    @initial: Sets the property to its default value
    @inherit: Inherits the property from the parent element
