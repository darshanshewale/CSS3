Pseudo-Classes

A pseudo class is CSS is used to define the special state of an element. It can be combined with a CSS Selector to add an effect to existing elements based on their states pseudo class names are not case sensitive

Hover pseudo class

this pseudo-class is used to add a special effect to an element when our mouse pointer is over it the below

color will changes if mouse hover over on same

Active pseudo classes

This pseudo-class is used to select an element that is activated when the user clicks on it

color will changes if click on me

Focus

used to select an element that is currently focused by the user it works on user input elements used in forms and is triggered as soon as the user click on it

visited pseudo class

This pseudo-class is used to select the links whic have been already visited by the user
