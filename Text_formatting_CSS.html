CSS Text formatting

        The CSS text formatting properties are used to format text,style text and perform different types of manipulation
        like word spacing,alignment,justification and text trasformation ,

Text Color

This property is used to set the color of the text and the color can be set by using a color name ,hex value , rgb value.

text-align

This property in CSS is used to specify the horizontal alignment of text in an element inside a block element or table cell box
        1)left:left aligned with respect to the container
        2)right:right aligned with respect to the container
        3)center:center aligned with respect to container
        4)justify : last line gets justified the last line will occupy the entire width of the container ,extra space in inserted between  the 
        words to achieve this property i.e can be set inter-word, inter-character,auto,none.
        5)start: left aligned if text direction is left to right 
        6)end:right aligned if text direction is left to right 
        7)auto:to be aligned as per text-aligned property of the container
        8)initial: to defualt
        9)aligned property of its parent element

CSS text-shadow property

 The text-shadow property in CSS is used to add shadows to the text.This property accepts a list of shadows
        that are to be applied to text,separated by the comma the default value of the text-shadow property is none.
        1)h-shadow: this property is required & used to specify the postion of horizontal shadow it accepts the negative values.
        2)v-shadow: this property is required & used to specify the positon of vetical shadow it is also accepts the negative values.
        3)blur radius: oit used to set the blur radius its default value  is 0 & is optional
        4)none: default value
        5)color:color of the shadow it is optional
        6)initial: default value
        7)inherit:inherited from parent

CSS letter spacing property

        The letter spcacing property in CSS used to set the spacing behaviour between text character i.e increasing or decreasing the 
        space between character in a text
        1)Normal:The normal letter spacing for the current font i.e no extra space between characters this is default value
        2)length:this mode extra inter character spcae in addition to the defualt space between characters negative value are also allowed
        3) inital: this mode sets this property to its default value

CSS line -height property

        The line-height property in CSS is used to set the amount of space used for lines such as in the text.Negative values are not allowed
        1)normal:normal line height . default value
        2)initial: default value
        3)number:this value is a unitless number multiplied by the current font size to set the line height in most cases
        4)length: mode a fixed line height is specified
        5)percentage:used to set line height in percent of the current font size.

CSS Direction property

        The direction property specifies the direction of the text writing within any block element it can have two values rtl(right to left)
        or ltr (left to right)
        1)rtl:right to left direction
        2)ltr:left to right direction
        3)initial:default Property
        4)inherit;inherits the property from the parent element

CSS word -spacing property

        1)Normal: it defines normal space between words which is 0.25em this is default value
        2)length: it defines an additonal space between words negative value are also allowed
