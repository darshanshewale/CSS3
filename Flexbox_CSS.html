Flexbox

    The flexbox or flexible box model in CSS is a one-dimensional layout model that has flexible and efficient layouts with distributed spaces
    among items to control their alignment structure ie it is a layout model that provides an easy and clean way to arrange item within a container
    flexbox can be useful for ceating small scale layouts & is responsive and mobile friendly 

    features of flexbox
    @A lot of flexibilty is given
    @arrangment & alignment of items
    @Order & sequencing of items
    @Bootstrap 4 is built on top of the flex layout

    before the flexbox model we had 4 layout modes
    @block: it is used to make sections in web pages
    @Inline : it is used for text 
    @table : it is used for dimensional table data
    @positioned : it is used for the explicit position of an element 

    There are 2 main componenets of the flexbox
    Flex container: The parent div which contains various devisions is called a flex container
    flex items: The items inside the container "div" are flex items

Flexbox Axes


    while working with flexbox we deal with 2 Axes
    @main Axis
    @Cross Axis
     
    main Axis
    By Default the main axis runs from left tp right 
    @Main Start: The start of the main axis is called Main start
    @Main size : The length between main start and main end is called main size
    @main end : The endpoint is called main end 
    main and cross axis

    left to right 
    @flex-direction:row;

    right to left
    @flex-direction:row reverse

    top to bottom
    flex-direction:column 

    bottom to top
    flex-direction:column-reverse 

    Cross Axis:The Cross axis will be perpendicular to the main axis

    Cross Axis: the cross Axis will be perpendicular to the main axis
    by Default cross axis runs perpendicular to the main axis i.e from top to bottom

    cross start: start of the cross axis is called cross start
    cross size: th length between cross start and cross end is called cross size
    cross end: the endpoint is called cross end

Properties of flexbox


        
        Prent properties 
         @display: define a flex container - flex formatting context
         @flex-direction: defines the main axis insides the container 
         @flex-wrap: allow flex items to wrap onto more than one line 
         @flex-flow:shorthand for flex-direction +flex-wrap 
         @justify-content: allow items to align along main axis
         @align-content: allow items to align along cross axis in a single line
         @align-items:align multiple lines of items on the cross axis

         Children/flex-items properties
         @Order: allows the change of order of items without altering the source order
         @flex-grow : allows an item to fill up the abvailable free space 
         @flex-shrink : allows an item to shrink it there is no enough free space available 
         @flex-basis: defines the size of an item before space is distributed
         @flex:it the shorthand for flex-grow +flex-shrink +flex-basis
         @flex-self: has the ability to align one single item within the flex container
