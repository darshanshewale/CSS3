CSS Links

A link is a connection from one page to another web page .CSS property can be used to style the links in various differnet ways 
        Sates of Link: before discussing CSS properties It is Important to know the states of link 
        Link can exist in different states and they can be styled using pseudo-classes

        There are four states of link given below 
        1)a.link: this is normal unvisited link.
        2)a.visited : this is a link visited by user at least once.
        3)a.hover : This is link when mouse hovers over it.
        4)a.active : this is link that is just clicked.
        
        
            a.link 
            {
                color:color_name;
            }
