CSS colors

        CSS Colors is used to set the colors of the HTML elememnts. this property is used to set font, colors,background colors etc
        Thw color of an element can be defined in the following ways:
        @Built-In color
        @RGB Format
        @RGBA Format
        @Hexadecimal Notation
        @HSL
        @HSLA
a)Built-In-Color:these are set of predefined colors while are used by its name
b)RGB Format:The RGB(red,Blue,Green)Format is used to define the color of an HTML element by specifying the RGB values range between 0 to 255 e.g Red(255,0,0)
c)RGBA FormatThe RGBA Format is simlar to the RGB bu the difference is RGBA contains A(Alpha)which specifies the transparency of elements. The value of alpha lies between 0.0 to 1.0 where 0.0 reprsents fully transparent and 1.0 reprsents not transparent.
d)Hexadecimal Notation:The hexadecimal Notation begins with # symbol followed by 6 chracter each ranging from 0 to F for example : red #FF0000,Green #00FF00 , Blue #0000FF etc
E)HSLHSL Stand for Hue,Saturation and Lightness respectively. This Format uses the cylindrical coordinate system. 
    @Hue is the degree of the color wheel its value lies between 0 to 360 where 0 represents red ,120 represents green and 240 represents blue colors
    @Saturation: It takes percentage value, where 100% represents completely saturated,while 0% reprsents completely unsaturated(gray).
    @Lightness: Itr takes percentage value where 100% reprsents white, while 0% represents black.

F)HSLAThe HSLA color property is similar to HSL property, but the difference is HSLA contains A(Alpha)which Specifies the transparency of elements. The value of alpha lies between 0.0 and 1.0 whrer 0.0 represents fully transparency and 1.0 represents not transparent
