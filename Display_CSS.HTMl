CSS Display

        The Display property in CSS defines the componenets (div,hyperlink ,headings etc) going to be placed on the web page As the 
        name suggest this property is used to define the display of the different parts of a web page.

        
    1)Inline: Dislpay an inline element 
    2)block: Display an block element 
    3)contents : used to disappear the container 
    4)flex : Display an element as a block0-level felx container
    5)Grid : Display an element as a block-level grid container 
    6)inline-block,inline-flex,inline-grid,inline-table 
    7)table,table-caption,table-column,table-header,table-footer,table-row,table-cell,table-column,table-row 
    8)None :used to remove for all element
    9)initial:it is used to set the default value
    10)inherits: it is used to inherits property from its parents elememnts
