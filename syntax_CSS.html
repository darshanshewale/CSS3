CSS syntax

A CSS syntax rule consist of a selector,property and its value. The selector points to the html element where the css style is 
        to be applied.  The CSS property is separated by semicolons it is combination of the selctor name followed by the property:value
        pair that is defined for the specific selector

syntax: selctor{property:value;}

        Declaration:A combination of a property and its corresponding value
        Selector:Used to target and select specific HTML elements to apply style to.
        Property :Defines the specific aspect or chracteristic of an element that you want to modify
        valueAssigned setting or parameter for a given property detrmining how the selected element should appear or behave.

h1{ color:green; text-align:center; }
