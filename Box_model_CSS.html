CSS box model

        CSS box model is a container that contains multiple properties including borders ,margins , padding and the content itsel 
        it is used to create the design and layout of webpages it can be used as a toolkit for customization the layout of different elements
        The web browsers renders every element as a rectangular box accrording to the CSS box model

        content Area: This area consists of cotent like text,images or other media content it is bounded by the content edge 
        and its dimensions are given by content-box width and height 

        padding area: It includes the elements padding This area is actually the space around the content area and within the border-box 
        its dimensions are given by the width of the padding box and the height of the padding box

        Border Area: It is the area between the box padding and margin its dimensions are given by the width and height of the border-box

        margin area:This area consists of space between the border and the margin the dimensions of the margin area are the margin box width around the 
        margin-box height it is useful to separate the element from its neighbors
