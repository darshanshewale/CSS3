CSS Outline

The Outline CSS Shortand property allows drawing a line around the element,outside the border.it is used to set the all the properties can be categorized into 4 type namely outline-style, outlin-color,outline-width & outline-offset .

Outline Style

it is used to set the appearance of the outline of an element, it tells us the style or type of outlin any other outline property can not be accessed without setting the outline style if absent then the default style wll be none.

Outline color

it is used to set the outline color of an element the color can be set by its name ie rgb value or hex value etc if absent then the default color will be the current Color

Outline Width

It is used to specify the width of this outline for a specifc element

CSS outline-offset

The CSS outline offset property set the amount of space between an outline and the edge or border of an element An Outline is a line drawn around the element outside the border edge the space betweeen the element and its outline is transparent. Also the outline may be non-rectangular.The default value is 0.
